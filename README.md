### 🚀 About Me

I build backend systems focused on real-time communication, authentication,  
and scalable data architecture — currently a Computer Systems Engineering  
student at MUET, and freelancing under **UmadTech**.  
**Stack:** Python · Django · Django REST Framework · PostgreSQL · Redis ·  
WebSockets · JWT Auth · Docker  
**Recent work:**  
- 🗨️ **RelayChat** — WhatsApp-style chat app with Django Channels, Redis,  
  JWT auth, presence tracking, and soft-delete patterns (15+ automated tests)  
- 📊 **ClientPulse** — Django CRM built around generic relations, signals,  
  custom model managers, and object-level permissions  
- 🔌 **Blog REST API** — Full CRUD API with JWT auth, file uploads, and  
  Railway deployment config  
Currently going deeper into system design, query optimization, and  
distributed backend patterns.

🔭 &nbsp;I'm currently working on **### Cortex  A Django-based backend layer for integrating LLMs (Gemini, Claude, GPT)  into real-world application workflows — beyond simple API wrapper calls.  **Core focus:** - 🧠 Structured LLM output handling (JSON schema validation, retries on    malformed responses) - 🔐 Secure API key management &amp; rate-limited request queuing - 📦 Reusable Django app for prompt templates, versioning, and response caching - ⚡ Async task handling for LLM calls (Celery + Redis) to avoid blocking    request/response cycles - 🔄 Tool-calling / function-calling integration between LLM outputs and    Django models  **Stack:** Django · DRF · Celery · Redis · PostgreSQL · Gemini API / Claude API  **Why:** Most LLM-backend integrations are thin wrappers around an API  call. Cortex is an attempt to treat LLM calls as first-class backend  operations — with proper error handling, observability, caching, and  async execution — the same way you'd treat any other external service  dependency.**  
🌱 &nbsp;I'm currently learning **System design, PostgreSQL internals &amp; query optimization, and LLM-backend integration patterns.**  
👯 &nbsp;I'm looking to collaborate on **Open-source Django packages, backend tooling, and real-time systems.**  
🤔 &nbsp;I'm looking for help with **Scaling WebSocket infrastructure and structuring async LLM calls in production Django apps.**  
💬 &nbsp;Ask me about **Django, DRF, PostgreSQL, Redis, JWT auth, WebSockets, and backend system design.**  
😄 &nbsp;Pronouns: **he/him**
